## PROJECT DESCRIPTION
Projek ini merupakan pengembangan aplikasi mobile berbasis Flutter yang berfokus pada pencarian dan penelusuran daftar anime. Aplikasi ini dirancang untuk memudahkan pengguna dalam menemukan berbagai judul anime lengkap dengan informasi detail seperti judul, deskripsi, rating, serta data pendukung lainnya.

Dalam proses pengembangannya, beberapa fitur utama telah berhasil diimplementasikan, antara lain:

- Autentikasi Pengguna
Pengguna dapat masuk ke aplikasi menggunakan akun Google melalui integrasi Firebase Authentication, atau memilih untuk membuat akun baru secara manual. Fitur ini memastikan proses login yang aman dan praktis.

- Fitur Pencarian (Search)
Aplikasi menyediakan kolom pencarian yang memungkinkan pengguna menemukan anime berdasarkan nama. Pencarian dirancang responsif sehingga hasil dapat ditampilkan secara cepat dan akurat.

- Integrasi Firebase
Firebase digunakan sebagai backend utama untuk menangani kebutuhan autentikasi, penyimpanan data, dan manajemen pengguna. Dengan Firebase, aplikasi menjadi lebih stabil dan mudah dikembangkan.

- Logout
Fitur untuk keluar dari akun juga telah disiapkan, memungkinkan pengguna mengganti akun atau mengakhiri sesi dengan aman.

Selain fitur-fitur tersebut, struktur aplikasi sudah dibangun menggunakan prinsip desain modern Flutter, termasuk pemanfaatan state management, widget yang terorganisir, dan arsitektur yang memudahkan pemeliharaan maupun pengembangan berkelanjutan. Ke depannya, aplikasi ini masih berpotensi dikembangkan lebih lanjut, seperti menambahkan fitur favorit, rekomendasi anime, hingga integrasi API tambahan untuk memperkaya data.
